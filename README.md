## Getting Started

### Prerequisites
* JAVA 15
* NodeJS

# API

## Installation
Build with Gradle <br>
* Windows Build
 ```sh
  .\gradlew build
  ```
  * Go into /build/libs and copy DashBoard-1.0-SNAPSHOT.jar
   ```sh
  cd .\build\libs && cp DashBoard-1.0-SNAPSHOT.jar
  ```
  * Linux Build
 ```sh
  ./gradlew build
  ```
  * Go into /build/libs and copy DashBoard-1.0-SNAPSHOT.jar
   ```sh
  cd ./build/libs && cp DashBoard-1.0-SNAPSHOT.jar
  ```
## Run
```sh
java -jar DashBoard-1.0-SNAPSHOT.jar
  ```
